# 롯데자이언츠 3000만 관중 기념 포트폴리오 프로젝트

2024년 롯데자이언츠 3000만 관중 기념 이벤트 기획 프로젝트 포트폴리오 웹사이트입니다.

## 📋 프로젝트 개요

롯데자이언츠 3000만 관중 달성을 기념하여 진행한 이벤트 기획 프로젝트로, **PPT·기획팀**으로서 팀원 의견을 반영하여 전략을 수립하고 데이터 분석을 토대로 전략을 제안하여 성공적으로 프로젝트를 리딩했습니다.

**수상 내역**: 우수상

## 🎯 프로젝트 배경

### 구단 요청사항
- 3000만 관중 기념인만큼 구단의 **역사**와 **낭만**을 담을 수 있는 기획
- **모든 연령대**가 참가할 수 있는 서비스 기획

### 프로젝트 목표
- 데이터 기반 전략 수립 및 서비스 기획
- 구단 인지도 상승 및 관중 수익 증가
- 이탈 관중 유지 및 신규 관중 유입

## 📊 데이터 분석 및 전략 수립

### 분석 내용
- **지난 5년간의 관중 데이터 분석**
  - 연령대별 분석
  - 성별 분석
  - 좌석 점유율 분석
  
- **구단 SNS 채널 버즈량 분석**
  - 핵심 타겟 설정
  - 이탈 위험 세대 파악 (40-50세대)
  - 화제성 향상 전략 방향 수립

### 핵심 발견
- 40-50세대가 가족단위 고객임을 확인
- 해당 세대의 이탈 위험 입증
- 사전 이벤트를 통한 화제성 제고 필요성 확인

## 🚢 이벤트 컨셉: Giantsfan-ship (삼범선)

부산과 항해, 그리고 롯데자이언츠의 상징성을 담은 **삼범선** 컨셉으로 전체 이벤트를 구성했습니다.

### 이벤트 구성

#### 📍 사전 이벤트
- **3000만 뽑기**: 전 연령대가 참여할 수 있는 참여형 이벤트
- **명예선원따라잡기**: 화제성 향상을 위한 전략 이벤트

#### ⚓ 본 이벤트

**1. 출항 (경기 시작 전)**
- **선장 인사**: 3000만 관중 달성 감사 인사로 브랜드 충성도 제고
- **나도 롯데 선원**: 전 연령대 참여형 메인 이벤트
- **돛을 펼쳐라**: 3000만 달성의 의미가 구단과 팬이 하나되었음을 시사

**2. 항해일지 (경기 진행 중)**
- **선원들의 이야기**: 롯데 구단과 얽힌 추억 공유로 소속감 고취
- **팬들의 항해사진**: 사전 이벤트와 연계한 메인 이벤트

**3. 바람 (경기 후반 및 종료 후)**
- **롯데 자이언츠를 향한 바람**: 동음이의어를 활용한 참여형 이벤트
- **팬들의 바람은 깃발을 타고**: 응원 단합력과 열정을 통한 소속감 고취

#### 🎁 추가 이벤트
- Giantsfan-ship을 구출하라
- 3천만예요
- 모스부호를 번역하라
- Giantsfan-ship 퀴즈

## 💰 예산 및 기대 효과

### 예산
- **총 예상 비용**: 67,590,680원
- 각 이벤트별 예산 가편성 분석 포함

### 기대 효과
1. **검색량 및 화제성 증가**로 구단 인지도 상승
2. 이벤트 기획으로 이탈하는 관중들을 붙잡고 **구단 관중 수익 증가**
3. **ROES 391%** 달성 (이벤트 예상 비용의 3.91배 수익 예상)
   - *ROES: 이벤트 중심 조직에 최적화된 ROAS의 확장형 지표*

## 💬 구단 피드백

### 긍정적 피드백
- 기존 컨셉에서 이어지는 **전략 기획**
- **수치 및 데이터 통한 분석**에 대한 긍정적 평가

### 개선 사항
- 대표 키워드인 3000만을 강조하는 이벤트 부족
- 기념 이벤트가 아닌 개막 이벤트 느낌을 받을 우려

# 웹 페이지 기능


## 🎯 주요 기능

- **반응형 웹 디자인**: 모든 디바이스에서 최적화된 레이아웃
- **인터랙티브 갤러리**: 
  - 데이터 및 현황 분석 섹션의 슬라이드 갤러리
  - 이벤트 이미지 lightbox 확대 기능
- **스크롤 애니메이션**: 부드러운 스크롤 및 페이드인 효과
- **이벤트 카드 인터랙션**: 
  - Items 섹션의 카드 클릭 시 해당 이미지 lightbox로 표시
  - Hover 효과 및 전환 애니메이션
- **Contact Form**: 연락처 정보 및 폼 기능

## 📁 프로젝트 구조

```
portfolio_lotte/
├── index.html              # 메인 HTML 파일
├── assets/
│   ├── css/                # 스타일시트
│   │   ├── main.css        # 메인 CSS
│   │   └── noscript.css    # NoScript 대체 CSS
│   ├── js/                 # JavaScript 파일
│   │   ├── main.js         # 메인 JavaScript
│   │   ├── gallery-style3.js    # 갤러리 슬라이드 기능
│   │   └── event-box-lightbox.js # 이벤트 카드 lightbox 기능
│   └── sass/               # SCSS 소스 파일
└── images/                 # 이미지 리소스
    ├── gallery/            # 데이터 분석 이미지
    ├── preEvent/           # 사전 이벤트 이미지
    ├── main_1/             # 본 이벤트 (출항) 이미지
    ├── main_2/             # 본 이벤트 (항해일지) 이미지
    ├── main_3/             # 본 이벤트 (바람) 이미지
    └── plusEvent/          # 추가 이벤트 이미지
```

## 🚀 시작하기

### 로컬에서 실행하기

1. 저장소 클론
```bash
git clone https://github.com/jseongeon/portfolio_lotte.git
cd portfolio_lotte
```

2. 웹 서버 실행
   - Live Server (VS Code 확장 프로그램) 사용
   - 또는 Python: `python -m http.server 8000`
   - 또는 Node.js: `npx http-server`

3. 브라우저에서 열기

## 🎨 웹사이트 주요 섹션

1. **배너**: 프로젝트 소개 및 시작
2. **구단 요청사항**: 프로젝트 배경 및 요구사항
3. **데이터 및 현황 분석**: 슬라이드 갤러리로 데이터 시각화
   - 버즈량 분석
   - 관중 데이터 분석
   - 전략 수립 과정
4. **구단 피드백**: 
   - 긍정적 피드백 (전략 기획, 데이터 분석)
   - 개선 사항 (3000만 키워드 강조 필요)
5. **이벤트 레이아웃**: Giantsfan-ship 이벤트 구조 시각화
   - 사전 이벤트: 3000만 뽑기, 명예선원따라잡기
   - 본 이벤트: 출항, 항해일지, 바람
6. **사전 이벤트 상세**: 사전 이벤트 기획안
7. **본 이벤트 상세**: 
   - 출항: 선장 인사, 나도 롯데 선원, 돛을 펼쳐라
   - 항해일지: 선원들의 이야기, 팬들의 항해사진
   - 바람: 롯데 자이언츠를 향한 바람, 팬들의 바람은 깃발을 타고
8. **추가 이벤트**: 추가 이닝 이벤트 기획안
9. **예산 가편성**: 예산 분석 및 가편성안 (67,590,680원)
10. **기대 효과**: 기대 효과 및 ROES 391% 분석
11. **Contact**: 연락처 정보 및 포트폴리오 링크

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 애니메이션
- **JavaScript (jQuery)**: 인터랙티브 기능
- **SCSS**: CSS 전처리기
- **Font Awesome**: 아이콘

## 📝 주요 기능 설명

### 갤러리 슬라이드 (Style3)
- 좌우 화살표 버튼으로 이미지 전환
- 부드러운 페이드 전환 효과
- 자동 높이 조정

### 이벤트 카드 Lightbox
- Items 섹션의 카드를 클릭하면 해당 이미지가 lightbox로 표시
- ESC 키 또는 배경 클릭으로 닫기

### Contact Form 아이콘
- Hover 시 아이콘이 텍스트로 전환
- 아이콘 확대 효과
